#通用解释器内核
此解释器采用C++编写
目前实现了表达式计算和函数注册与调用
关键字注册符号识别等
目前只支持双目运算符
#发展
实现了for into循环
实现了RunBlock函数
#未来展望
预计将修改runExpression函数
统一使用getExp函数获取单词
以方便未来维护和控制
同时getExp可以保证自动跳过前后空格
减少BUG发生率
